JPA (Java Persistence API) là một đặc tả (specification) trong Java dùng để quản lý dữ liệu giữa các đối tượng Java (entity) và cơ sở dữ liệu quan hệ. Nó cung cấp một cách tiếp cận hướng đối tượng để truy xuất, lưu trữ và quản lý dữ liệu, giúp lập trình viên không cần viết nhiều câu lệnh SQL thuần. JPA hoạt động thông qua các provider như Hibernate, EclipseLink,… giúp ánh xạ (mapping) các class Java sang bảng dữ liệu một cách tự động và linh hoạt. JPA quan trọng vì nó chuẩn hóa cách xử lý dữ liệu trong Java, giảm thiểu lỗi khi truy xuất dữ liệu, và cải thiện khả năng bảo trì cũng như tái sử dụng mã nguồn. Nhờ khả năng trừu tượng hóa, lập trình viên có thể tập trung vào logic nghiệp vụ mà không bị ràng buộc bởi chi tiết kỹ thuật của từng hệ quản trị cơ sở dữ liệu.
